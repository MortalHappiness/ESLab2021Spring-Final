## Build frontend

```shell
cd frontend
npm install
npm run build
```

## Start Server

```shell
cd server
npm install

# development
npm start

# production
npm run production
```

### Environment variables (And their default value)

```
SERVER_PORT=8000
SOCKET_PORT=8001
```
